Privacy Policy

Last Updated: June 14, 2026
🛡️ Zero Data Collection Policy: This extension is built with a local-first design philosophy. None of your session data, conversations, or account configurations are sent to external servers or third parties. All processing happens entirely on your machine.

This Privacy Policy describes how the "ChatGPT Bulk Delete Manager" Chrome Extension handles your data. We take user privacy extremely seriously and have engineered this tool to run completely locally inside your browser sandbox.

1. Information We Collect

We do not collect any personal information, usage metrics, analytics, or private tokens. The extension operates with a zero-telemetry architecture. We do not have databases or remote tracking servers.

2. Data Processing and Usage

To enable multi-selection, live previews, backups, and bulk deletion, the extension must interact locally with ChatGPT's internal backend endpoints on the page. Here is how that data is processed:

  • Session Access Tokens: The extension queries your active browser session locally on chatgpt.com to acquire the session bearer token. This is used solely to authenticate pagination lists and deletion requests, and never leaves your browser.
  • Conversation History List: The extension retrieves your chat list dynamically to render the dashboard checkboxes. This history is stored in temporary runtime memory and is destroyed when you close the manager modal.
  • Dialogue Previews: Dialogue content is loaded in real-time only when you click a card to review it, and is rendered directly inside your tab's DOM tree.
  • Local Backups: Dialogue text is converted locally in your browser to a Markdown file download. We do not store or mirror these backups.

3. Extension Permissions Justification

The extension requires the following permissions in Google Chrome to perform its tasks:

  • activeTab: Used to inject the dashboard UI (CSS styles and script elements) on top of the ChatGPT tab.
  • Host Permissions (https://chatgpt.com/* & https://chat.openai.com/*): Required to allow the extension to read and write visibility states (deletion requests) for your active web sessions on those domains.

4. Third-Party Services

This extension interacts with OpenAI's ChatGPT web interface. However, it is an independent developer tool and is not affiliated with, endorsed by, or sponsored by OpenAI. Any communication with OpenAI's backend APIs is subject to OpenAI's own Privacy Policy and Terms of Service.

5. Changes to This Policy

Since the project is open-source, any updates to the extension's behavior or privacy declarations will be logged and visible in our public GitHub repository commits.

6. Contact & Support

If you have any questions or would like to review the code yourself, please visit the official project repository at GitHub.
